# arithmetic-bot #

`ab` is an arithmetic formula generate bot.

Generate expirement by level, for example: 

```shell
> ab exp -l=1                        
4 + 3 =                 4 + 1 =                 6 + 4 =  
9 + 0 =                 7 + 1 =                 6 + 2 =  
3 + 2 =                 8 + 2 =                 0 + 0 =  
10 + 0 =                2 + 2 =                 4 + 4 =  
8 + 0 =                 8 + 2 =                 5 + 4 =  
6 + 1 =                 4 + 0 =                 6 + 1 =  
6 + 2 =                 7 + 1 =                       
```

Generate addition formula, for example: 

```shell
> ab plus -n=10 -s=1000 
41 + 220 =              762 + 93 =              242 + 227 =  
954 + 46 =              551 + 166 =             690 + 162 =  
145 + 303 =             579 + 105 =             701 + 77 =  
840 + 99 =              
```
